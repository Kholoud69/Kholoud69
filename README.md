## ⚙️ What I Do – Data Engineering

As a **Data Engineer**, I focus on building the foundations that make data useful and accessible. My responsibilities include:

- **Building Data Pipelines:** Extract, transform, and load (ETL) data from multiple sources into warehouses or data lakes.  
- **Managing Databases & Storage:** Design and maintain relational (MySQL, PostgreSQL) and NoSQL databases, as well as data warehouses and data lakes.  
- **Ensuring Data Quality:** Clean, validate, and monitor data to make sure it’s accurate, complete, and reliable.  
- **Big Data & Streaming:** Work with large-scale data using tools like Apache Spark, Hadoop, and real-time streaming platforms such as Kafka.  
- **Automation & Workflow Orchestration:** Schedule and monitor data workflows using Apache Airflow, Luigi, or similar tools.  
- **Collaboration:** Provide clean and organized data for analysts and data scientists to derive insights and build dashboards.  

> In short, I design, build, and maintain the pipelines that turn raw data into actionable insights
